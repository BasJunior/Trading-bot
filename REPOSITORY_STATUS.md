## 🛠️ Getting Started

### For New Users:
```bash
# Clone the repository
git clone https://github.com/yourusername/deriv-telegram-bot.git
cd deriv-telegram-bot

# Run automated setup
python scripts/deploy.py

# Start the bot
./start_bot.sh
```

### For Production:
```bash
# Production deployment
sudo ./scripts/deploy_production.sh

# With Docker
docker-compose up -d
```
